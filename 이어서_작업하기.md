# 다른 PC에서 개발 이어서 하기 — 핸드오프 가이드

이 문서는 **개발 환경을 다른 컴퓨터로 옮겨 이어서 작업**하기 위한 안내입니다.
(시연용 실행은 `README.md` / `대시보드 실행.bat` 참고)

---

## 1. 사전 준비 (새 PC에 설치)
- **Node.js LTS 20 이상** (권장 **24** — 모크 생성기가 `.ts`를 직접 실행, 타입 스트리핑 사용)
  - https://nodejs.org → LTS 설치
- **Git** (이력 이어가려면) — https://git-scm.com
- 코드 에디터 (VS Code 권장)

## 2. 프로젝트 가져오기 (둘 중 하나)
- **(A) zip 핸드오프:** `핸드오프_개발용.zip`(node_modules 제외, .git 포함)을 풀기
- **(B) Git 원격:** 원격 저장소가 있으면 `git clone <url>`

## 3. 개발 환경 셋업
```bash
cd fixed_broadband_quality_dashboard   # 폴더명은 상황에 맞게
npm install            # 의존성 설치 (node_modules 재생성)
npm run mock           # public/quality_data.json 생성 (zip에 포함돼 있으면 생략 가능)
npm run dev            # http://localhost:5173
```
기타: `npm run build`(타입체크+빌드), `npm run typecheck`

## 4. 기술 스택
- **React 18 + Vite 5 + TypeScript** (모든 소스 `.ts/.tsx`, import는 명시적 확장자)
- **ApexCharts**(react-apexcharts) — 차트
- **html2canvas** — 화면 스크린샷(동적 import)
- tsconfig: `allowImportingTsExtensions` + `noEmit`, `moduleResolution: bundler`

## 5. 데이터 구조 (중요)
- 프론트는 **단일 정적 파일** `public/quality_data.json` 만 fetch (라이브 API 직접호출 없음).
- **다중 해상도(티어)** 한 파일:
  - `fine` 10분/2일(실제 원샘플 FR-03 절단) · `mid` 1시간/30일 · `coarse` 1일/365일(대표 집계)
  - 범위 선택 → 티어 자동, 버킷(10분/1시간/1일)은 티어 기본 이상만
- 생성기: `scripts/generate-mock.ts` (현재 **시뮬레이션** 데이터)

## 6. 디렉터리 맵
```
src/
  App.tsx                  앱 셸/상태(출처·기간·버킷·테마·초기화)

  theme.ts                 테마 영속 + ISP 색상
  types.ts                 공용 타입
  data/

    metrics.ts             지표 카탈로그(출처/단위/하드임계값)
    quality.ts             quality_data.json 로더 + 티어 디코딩
  lib/
    stats.ts               FR-03 절단평균/저표본/하드필터 (생성기와 공유)
    aggregate.ts           티어→버킷 재집계
    peak.ts                FR-04 피크타임 분석

    screenshot.ts          html2canvas 캡처
  components/
    IspMultiSelect.tsx     통신사 멀티셀렉트(80vh, 접기)
    MetricChart.tsx        지표 차트(휠줌, 저표본 마커, 추적성 툴팁)
    MetricSection.tsx      차트+피크 한 묶음
    PeakTimeWidget.tsx     피크타임 카드
    ApiSettings.tsx        데이터소스/토큰 설정 모달
scripts/generate-mock.ts   단일 다중해상도 모크 생성기
.github/workflows/collect.yml  10분 크론(현재 모크 재생성, FR-07 골격)
```
